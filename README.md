# Helabooking-Infra Repo

Infrastructure as Code (IaC) and documentation repository for the helabooking project, enabling automated provisioning, configuration, and management of its onprem &amp; cloud infrastructure.

## Folder Structure

- `_docs`: Documentation related to setting up and managing the helabooking infrastructure.
- `on-prem`: On-premises infrastructure scripts and configurations.
- `cloud`: Remote/cloud infrastructure scripts and configurations.
